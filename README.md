# Online-Learning-Adaptive-Hedge-
Online Learning (Adaptive Hedge regret curve compared to that of Hedge with fixed learning rate)

In this paper, I compare the implementations of the Hedge and Adaptive Hedge algorithms in three different settings of synthetic losses. The deliberately diverse natures of synthetic losses provides an opportunity to study the behavior of these algorithms depending on the difficulty of the environment. This was achieved by analyzing the regret versus time plots for Adaptive Hedge algorithm with adaptive learning rate and Hedge algorithm with fixed learning rate. The results showed that AdaHedge adapts well to the difficulty of the problem: in "easy instances" it has smaller regret than Hedge while simultaneously guaranteeing worst case regret bounds.
